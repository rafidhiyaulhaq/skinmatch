Overview

SkinMatch adalah aplikasi rekomendasi skincare yang menganalisis tipe kulit pengguna melalui kuesioner dan memberikan rekomendasi produk yang dipersonalisasi dengan notifikasi email otomatis.

Fitur Utama:

  • Autentikasi pengguna (register/login)
  • Quiz analisis tipe kulit
  • Rekomendasi produk personalisasi
  • Manajemen profil pengguna
  • Pelacakan riwayat quiz
  • Notifikasi email otomatis dengan hasil analisis

Integrasi Sistem

FinTrackIt Email Service

Integrasi dengan layanan email FinTrackIt (David Dewanto, 18222027) untuk pengiriman hasil analisis.

Fitur:
  • • Notifikasi email otomatis setelah quiz selesai
  • • Template HTML dengan konten yang dipersonalisasi
  • • Autentikasi aman menggunakan API key
  • • Autorisasi berbasis token
  • • Rate limit handling
  • • Automatic token refresh

Integration Endpoints
