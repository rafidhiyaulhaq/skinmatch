SkinMatch Frontend Documentation

GitHub Repository: https://github.com/rafidhiyaulhaq/skinmatch

Overview

SkinMatch is a skincare recommendation application that analyzes user's skin type through a questionnaire and provides personalized product recommendations with automatic email notifications.

Live URL: https://skinmatch-five.vercel.app

Features

  • User authentication (register/login)
  • Skin type analysis quiz
  • Personalized product recommendations
  • User profile management
  • Quiz history tracking
  • Automatic email notifications with analysis results
  • Integrated email service for result delivery

Tech Stack

Frontend:

  • React 18
  • Vite
  • React Router v7
  • Tailwind CSS
  • Axios

Backend & Services:

  • Node.js
  • Express
  • MongoDB
  • JWT Authentication
  • FinTrackIt Email Service Integration

Installation

# Clone repository
git clone https://github.com/rafidhiyaulhaq/skinmatch.git

# Frontend setup
cd client
npm install
npm run dev

# Backend setup
cd ../server
npm install
npm run dev

Pages & Routes

Public Routes

  • / - Home page
  • /login - User login
  • /register - User registration

Protected Routes

  • /quiz - Skin analysis questionnaire
  • /results - Analysis results & recommendations
  • /profile - User profile management

API Integration

Backend API Documentation: https://skinmatch.up.railway.app/docs

Available services (src/services/api.js):

  • Authentication (register, login, profile)
  • Quiz submission and history
  • Product recommendations
  • Email notifications (integrated with FinTrackIt)

External Service Integration

FinTrackIt Email Service (by David Dewanto, 18222027)

Integrated email service for sending personalized quiz results and product recommendations to users.

  • Automatic email notifications after quiz completion
  • HTML email templates with personalized content
  • Secure authentication using API keys
  • Token-based authorization for API requests
  • Integration with user quiz flow
  • Automated result delivery system

Deployment

Frontend (Vercel)

  • URL: https://skinmatch-five.vercel.app
  • Build Command: npm run build
  • Output Directory: dist
  • Environment Variables:
    • VITE_API_URL
    • VITE_FINTRACKIT_API_KEY

Backend (Railway)

  • URL: https://skinmatch.up.railway.app
  • Start Command: npm start
  • Environment: Node.js
  • Environment Variables:
    • MONGODB_URI
    • JWT_SECRET
    • FINTRACKIT_API_KEY
