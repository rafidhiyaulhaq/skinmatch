Overview

SkinMatch adalah aplikasi rekomendasi skincare yang menganalisis tipe kulit pengguna melalui kuesioner dan memberikan rekomendasi produk yang dipersonalisasi dengan notifikasi email otomatis.

Fitur Utama:

  • Autentikasi pengguna (register/login)
  • Quiz analisis tipe kulit
  • Rekomendasi produk personalisasi
  • Manajemen profil pengguna
  • Pelacakan riwayat quiz
  • Notifikasi email otomatis dengan hasil analisis

  • Backend container (Node.js) - Dijalankan di Railway
  • Database container (MongoDB) - MongoDB Atlas untuk production

Container Orchestration:

  • Development: Docker Compose untuk menjalankan semua service
  • Production: Railway Container Service untuk backend
  • Automatic deployment saat ada push ke main branch

API Documentation

Authentication

Register

POST /api/auth/register

{
    "username": "string",
    "email": "string",
    "password": "string"
}

Login

POST /api/auth/login

{
    "email": "string",
    "password": "string"
}

Get Profile

GET /api/auth/profile

Headers: Authorization: Bearer token

Quiz

Submit Quiz

POST /api/quiz/submit

{
    "answers": {
        "oiliness": "string",
        "dryness": "string",
        "sensitivity": "string",
        "pores": "string",
        "acne": "string"
    }
}

Get Quiz History

GET /api/quiz/history

Headers: Authorization: Bearer token

Products

Get Recommendations

GET /api/products/recommendations?skinType=string

Headers: Authorization: Bearer token

Get All Products

GET /api/products

Query Parameters: category, brand, minPrice, maxPrice (all optional)

Integrasi Sistem

FinTrackIt Email Service

Integrasi dengan layanan email FinTrackIt (David Dewanto, 18222027) untuk pengiriman hasil analisis.

Fitur:
  • Notifikasi email otomatis setelah quiz selesai
  • Template HTML dengan konten yang dipersonalisasi
  • Autentikasi aman menggunakan API key
  • Autorisasi berbasis token
  • Rate limit handling
  • Automatic token refresh
