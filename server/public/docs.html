SkinMatch API Documentation

Base URL

https://skinmatch.up.railway.app

Authentication Endpoints

Register

URL: /api/auth/register

Method: POST

Request Body: 
{
 "username": "string",
 "email": "string",
 "password": "string"
}

Login

URL: /api/auth/login

Method: POST

Request Body: 
{
 "email": "string",
 "password": "string"
}

Get Profile

URL: /api/auth/profile

Method: GET

Headers: Authorization: Bearer token

Quiz Endpoints

Submit Quiz

URL: /api/quiz/submit

Method: POST

Headers: Authorization: Bearer token

Request Body: 
{
 "answers": {
   "oiliness": "string",
   "dryness": "string",
   "sensitivity": "string",
   "pores": "string",
   "acne": "string"
 }
}

Get Quiz History

URL: /api/quiz/history

Method: GET

Headers: Authorization: Bearer token

Products Endpoints

Get Recommendations

URL: /api/products/recommendations

Method: GET

Headers: Authorization: Bearer token

Query: skinType=string

Get All Products

URL: /api/products

Method: GET

Query Parameters:

  • category (optional)
  • brand (optional)
  • minPrice (optional)
  • maxPrice (optional)
